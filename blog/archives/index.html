Edit blog with Windows Live Writer

Aug 06 2013

由于每次写博客都通过html格式比较麻烦，所以此处推荐使用Windows Live Writer本地编辑。

        1.首先用Windows Live Writer编辑完成文章，选择以“源代码”形式查看,复制文本。

        2.在本地“octopress/public/blog/archives/index.html”找到需要更新的日志,通过记事本软件打开此html。

        3.找到<div class="entry-content"></div>，将复制的代码粘贴进去，保存关闭。

        4.在octopress目录下运行“rake deploy”命令就可以发布了。（发布之前可“rake preview”本地查看一下~）

备注：如有图片的话，一般做法还需将图片保存到对应文件夹。此处可以先在“博客园”等网站通过Windows Live Writer发布博客，再将源码复制。即省去图片调用。又可做到一篇博客不同地方迅速发布，保留备份。（未完待续...）
